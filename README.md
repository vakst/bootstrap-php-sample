Basic bootstrap application
======================

What inside?
---------

PHP, Twig, API, Cache, Bootstrap, SASS, Grunt

Installation
---------

1) Checkout project from GitHub

```bash
git clone https://github.com/vakst/bootstrap-php-sample.git ./
```

2) Install dependencies

```
composer install
npm install
```

3) Configure project

``
src/Resources/config.yml
``

4) Launch grunt

```
grunt buildcss
```
